**2022년 여름, 트위터에서 취향표 제작 열풍이 일어나면서부터 취향표 사이트를 만들어야겠다는 생각이 들었다.**

그래서 처음으로 **Flutter Web**을 이용해서 만들었던 것이 [**세븐틴 취향표 제작 사이트**](https://svt-style.web.app/)였다.

<Image src={imgSvtSite} alt="" />

제일 첫 번째로 나는, 여섯 문항에 대해 본인이 선택하고 싶은 멤버들을 선택할 수 있도록 만들었다.
그런 다음에 제일 중요한 게 "저장" 기능이라고 생각해서 저장 기능을 만들기 위해 노력했다.

하지만, Flutter Web에서 이미지 파일을 **저장하는 것이
특정 웹뷰, 특정 모바일 기기에서 되지 않아 거의 작동하지 않음**을 파악하였다.
그래서 "캡처해서 쓰세요!"라는 문구를 넣어 캡처를 통해 저장할 수 있게 안내했다.

여섯 문항 버전을 만들고 난 다음,
난 서비스에 대한 피드백을 듣고 힘을 얻는 개발자이니까!
트위터에 소소한 경품을 걸고 RT 이벤트를 열었다.
RT 이벤트의 힘은 놀라웠다.
2022년 8월 5일부터 2022년 12월 24일까지의 **트윗 노출 수는 22만, 링크 클릭 수는 1만 명**이나 됐다.

Flutter 개발을 주로 하고 있는 터라, Flutter Web으로 빠르게 개발해 보고 싶은 생각이 들었던 것이지만,
Flutter Web은 아직 내가 쓰고 싶은 분야들(생성기 사이트 등)에 쓰기엔 적절하지 않았다.
첫 번째로는 위에서 말했던 것처럼, 이미지 파일 저장이 잘 안 되었다.
두 번째로는, 트위터 유저들 특성 상 모바일에서 사이트 방문을 많이 하는데, 각기 다른 모바일에서 UI 구현에 있어서 overflow가 안 나도록 만드는 것이 힘들었다.

그래서 다음부터는 React로 다시 만들어 봐야겠다고 생각을 하고 까먹고 있었다.

그런데!
세븐틴만 보고 살 줄 알았던 내가 더보이즈에 입덕을 하게 될 줄은 몰랐다.
입시 준비로 바빴던 2018, 2019년에 잠깐 관심을 가지고 봤던 그룹이었고, 평소에도 노래는 자주 들었던 편이다. 근데 갑자기 입덕을 해 버리게 되었고, 더보이즈 관련한 프로젝트를 해 보고 싶었다.

그러다가 그냥 **Next.js와 Tailwind CSS, Typescript**를 약간씩 배우고 있었기 때문에 더보이즈 버전 취향표를 만들어 보자는 생각을 하게 됐다.

<Image src={imgTbzSite1} alt="" />
<Image src={imgTbzSite2} alt="" />

그래서 더보이즈 덕질 계정으로 만들어서 올렸던 게 [**더보이즈 취향표 제작 사이트**](https://tbz-style.vercel.app/)였다.
[**GitHub Repository 링크**](https://github.com/DevvTyga/tbz-style)

드디어 Flutter Web에서 힘들었던 이미지 저장 기능을 **html2canvas와 file-saver**를 이용해 만들었다. 근데, next에서 제공하는 Image를 사용하면 이미지를 저장했을 때 사진 사이즈가 깨져서 제대로 나오지 않아 그냥 img 태그를 써서 만들었다. 그게 좀 아쉬워서 다음에 이미지 저장이 필요할 때 사진이 들어가는 경우에는 이 부분을 파악해 봐야겠단 생각이 들었다.

웹 프론트엔드 개발은 거의 안 하기도 하고, 잘하는 편도 아니었기 때문에 초반에는 진짜 단순하게 만들었다.
또 아쉬웠던 점은, 각 항목에 대해 멤버를 선택하는 버튼, 선택하는 버튼을 눌렀을 때 나오는 팝업 모달이 반복됨에도 그냥 내용만 바꿔서 복사하고 붙여넣기 식으로 만들어서 배포도 하고, 트위터 이벤트도 열었다.

나는 진짜 불편함을 느껴 봐야 복사하고 붙여넣기 방식을 벗어나 컴포넌트, 레이아웃화를 하기 때문에!
친구 요청도 있었고 해서, 더보이즈 취향표 제작 사이트를 기반으로 하여 멤버 수가 많은 [**NCT 취향표 제작 사이트**](https://nct-style.vercel.app/)를 만들었다.

유닛도 다양하고, 전체적인 멤버도 많아서 복사 붙여넣기는 비효율적이라는 것을 몸소 느껴 보고, 컴포넌트화, 레이아웃화를 연습할 수 있었다. 그래서 NCT 취향표 제작 사이트를 만들고 나서 더보이즈 취향표 제작 사이트 프로젝트도 리팩토링을 진행하였다.

[**GitHub Repository 링크**](https://github.com/DevvTyga/nct-style)

이렇게 나름 웹 프론트엔드 개발을 천천히 연습해 보는 과정이 있었다.

Flutter Web ->
Next.js + Typescript + TailwindCSS (하지만 컴포넌트화와 레이아웃화는 하지 않은) ->
Next.js + Typescript + TailwindCSS (컴포넌트화와 레이아웃화를 한)

이렇게 해서 같은 주제의 프로젝트를 세 번이나 만들어 보면서 나름의 성장을 했고, 많이 뿌듯하다.
다음에도 덕질 관련 프로젝트를 해 볼 예정인데, 이번에 성장한 과정처럼 다음에도 많은 성장을 했으면 좋겠다!
